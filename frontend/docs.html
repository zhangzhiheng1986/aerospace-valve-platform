🚀 航空航天电磁阀研发平台

本平台是专为航空航天电磁阀研发设计的综合仿真分析平台，集成电磁优化、材料管理、流体仿真、热力学分析、结构强度评估等核心功能，为工程师提供一站式研发支持。

核心功能

⚙️ 电磁阀优化设计

基于粒子群优化算法(PSO)的多参数优化，自动寻找最佳线圈匝数、线径、电流等参数组合，最大化电磁力并满足温度约束。

🔬 专业材料数据库

包含21种航空航天级材料，涵盖导体、磁性、绝缘、结构、密封、弹性、涂层7大类，支持智能推荐和绕组参数计算。

🌊 流体仿真分析

基于CFD的阀门内部流场仿真，计算压力分布、流速分布、流量系数、空化风险等关键参数。

🌡️ 热力学分析

热传导、对流换热、辐射换热综合分析，温度场仿真，热应力计算，热时间常数评估。

🔧 结构强度分析

基于FEM的应力应变计算，安全系数评估，疲劳寿命预测，振动特性分析。

技术特点

  • 专业性强：面向航空航天领域，材料和算法均符合行业标准
  • 集成度高：多学科仿真分析一体化，无需切换工具
  • 易用性好：Web界面直观，参数输入便捷，结果可视化
  • 可扩展：模块化设计，便于功能扩展和定制

⚙️ 电磁阀优化设计

优化目标

在满足温度约束的前提下，最大化电磁力输出。

优化参数

参数 说明 默认范围
AWG线规 导线美国线规 20-30
匝数 线圈匝数 500-3000
电流 工作电流 (A) 0.1-2.0
内径 线圈内径 (mm) 5-20

约束条件

  • 最高温度 ≤ 180°C（默认）
  • 电流密度 ≤ 许用值
  • 填充系数 ≤ 0.7

优化算法

采用粒子群优化算法(PSO)，具有全局搜索能力强、收敛速度快的特点。

提示：优化结果仅供参考，实际设计需结合工程经验和实验验证。

材料属性

每种材料包含以下属性数据：

  • 基本信息：名称、类别、执行标准
  • 物理性能：密度、熔点
  • 电学性能：电阻率、电导率
  • 磁学性能：磁导率、饱和磁感应强度
  • 力学性能：抗拉强度、延伸率、硬度
  • 热学性能：热导率、热膨胀系数
  • 认证信息：航空航天等级、军标编号

智能推荐

🌊 流体仿真分析

分析类型

  • 稳态分析：计算稳定工况下的流场分布
  • 瞬态分析：模拟启闭过程中的流场变化

输出结果

  • 压力场分布
  • 流速场分布
  • 流量系数 (Cv/Kv)
  • 雷诺数和流动类型
  • 空化指数和风险评估
  • 压降和功率损耗

🌡️ 热力学分析

传热模式

  • 热传导：通过固体材料的热传递
  • 对流换热：流体与固体表面的热交换
  • 辐射换热：通过电磁波的热传递

分析功能

  • 稳态热分析
  • 瞬态热分析（温度随时间变化）
  • 2D温度场仿真
  • 热应力计算
  • 热时间常数计算

典型应用

计算电磁阀线圈工作温度、评估散热性能、预测热变形。

🔧 结构强度分析

分析内容

  • 应力分析：计算各种载荷下的应力分布
  • 变形分析：计算结构变形量
  • 安全系数：评估设计裕度
  • 疲劳寿命：预测循环载荷下的寿命
  • 振动特性：计算固有频率

失效准则

  • 最大应力准则
  • Von Mises屈服准则
  • Goodman疲劳准则

压力容器分析

针对阀体承压部分，计算环向应力、轴向应力、径向应力。

❓ 常见问题

Q1: 如何修改默认管理员密码？

登录后，在后台管理页面点击用户列表中的"编辑"按钮，或使用修改密码API。

Q2: 优化结果不准确怎么办？

检查输入参数是否合理，尝试调整优化算法参数（粒子数、迭代次数），或联系技术支持。

Q3: 如何添加新材料？

目前需要修改后端 materials_database.py 文件，后续版本将提供Web界面。

Q4: 仿真结果如何导出？

仿真结果以JSON格式返回，可复制保存或使用浏览器打印功能导出PDF。

Q5: 平台支持哪些浏览器？

推荐使用Chrome、Edge、Firefox最新版本。不支持IE浏览器。
