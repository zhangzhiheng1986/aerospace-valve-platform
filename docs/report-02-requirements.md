## 四、约束条件

1. **技术栈不变**: Python/Flask + Vanilla JS + MathJax（不引入React/Vue等框架，保持轻量）
2. **文件结构**: 保持Blueprint架构
3. **Windows兼容**: Python代码ASCII-only, HTML UTF-8
4. **零外部付费依赖**: 所有库开源免费
5. **GBK编码约束**: 后端代码无特殊Unicode字符
